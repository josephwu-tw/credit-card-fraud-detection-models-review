# 💳 Credit Card Fraud Detection Models: A Review

## 🚀 Overview
This repository contains a comprehensive review of deep learning methods for credit card fraud detection, based on my Master's thesis.

It analyzes and compares multiple machine learning and deep learning approaches for detecting fraudulent transactions in financial systems.

---

## 🎯 Problem
Credit card fraud detection is a critical challenge due to:

- Highly imbalanced datasets  
- Evolving fraud patterns  
- Non-linear relationships in transaction data  

Traditional rule-based systems are often insufficient to detect modern fraud behaviors.

---

## 🧠 Scope of Study

This work focuses on:

- Reviewing classical and deep learning models  
- Comparing model performance and limitations  
- Identifying key challenges in fraud detection systems  

---

## 📚 Methods Covered

### Machine Learning
- Logistic Regression  
- Decision Trees  
- Random Forest  
- Gradient Boosting  

### Deep Learning
- Artificial Neural Networks (ANN)  
- Recurrent Neural Networks (RNN / LSTM)  
- Autoencoders  

---

## ⚙️ Key Concepts

- Imbalanced data handling (SMOTE, undersampling)  
- Feature engineering for transaction data  
- Evaluation metrics (Precision, Recall, F1-score, ROC-AUC)  

---

## 📊 Insights

- Deep learning models show strong potential in capturing non-linear fraud patterns  
- Model performance depends heavily on data preprocessing and feature engineering  
- No single model dominates — trade-offs exist between interpretability and performance
